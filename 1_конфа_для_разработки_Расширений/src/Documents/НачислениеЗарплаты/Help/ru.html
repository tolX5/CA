Документ "Начисление зарплаты"

Документ предназначен для отражения начисления зарплаты, удержаний из заработной платы и отчислений с ФОТ.

При вводе документа в шапке нужно указать следующие реквизиты:

  • Подразделение  - подразделение организации, если необходимо начислить заработную плату по определенному подразделению.
  • Месяц начисления - календарный месяц, за который начисляется заработная плата.

Документ заполняется автоматически по кнопке Заполнить.

Для заполнения начислений важно, чтобы в информационной базе для одного из начислений  было назначено, что оно является "основным", например, начисление "Оклад" (категория начисления для него в списке всех начислений - "Повременная оплата труда"). То есть это то начисление, которое будет начисляться сотруднику ежемесячно за отработанное время. Именно это "основное начисление" должно быть указано в кадровом документе "Прием на работу" для каждого сотрудника. Остальные виды начислений (материальная помощь, больничный, отпуск текущего периода, надбавка, премия и пр.) создаются в конфигурации как прочие виды начислений и применяются в качестве дополнительных начислений к основному.

=====================================

Примечание по деноминации на 01.07.2016 года:

Для учета ограничения при расчете отчислений в ФСЗН за июль 2016 года в данном документе, необходимо в форме «Настройках учета зарплаты» указать предельные величины базы страховых взносов на 01.06.2016 года уже в проденоминированных суммах, т.к. они будут использоваться при расчете отчислений за июль 2016 года.
